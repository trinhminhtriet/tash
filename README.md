# tash = task + bash

🚀 **tash** – A fast, YAML-powered task runner for local & remote automation with SSH & Docker support.

## 🚀 Installation

Download from [latest releases ](https://github.com/trinhminhtriet/gloc/releases)

## 💡 Usage

### Basic Usage

```sh
make build
ln -s $(pwd)/bin/tash /usr/local/bin/tash
tash --version
```
